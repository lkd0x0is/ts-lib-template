# Installation

Welcome to `ts-lib`! To start using the Calculator library in your project, you can install it via your preferred package manager.

## Using pnpm (Recommended for this workspace)

If you are working within a pnpm workspace or prefer pnpm:

```bash
pnpm add ts-lib
```

## Using npm

```bash
npm install ts-lib
```

## Using yarn

```bash
yarn add ts-lib
```

Replace `ts-lib` with the actual published package name if it's scoped (e.g., `@your-scope/ts-lib`).

Once installed, you can import the `Calculator` and other utilities into your TypeScript or JavaScript project.
