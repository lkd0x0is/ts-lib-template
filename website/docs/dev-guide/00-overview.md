Welcome to the Developer Guide for the TypeScript Library Template!

This guide is designed to provide you with in-depth information about the project's structure, development workflows, tooling, and customization options. Whether you're a new contributor or looking to understand the template's architecture, these documents aim to be your comprehensive resource.

## What You'll Find Here

This guide is broken down into several key areas:

* **[Project Philosophy & Tech Choices](./01-philosophy-tech-choices.md):** Understand the reasoning behind the tools and methodologies used in this template.
* **[Detailed Project Structure](./02-project-structure-detailed.md):** A closer look at the directories and important files within the monorepo.
* **Tooling & Scripts:**
  * **[Helper Scripts (Advanced Usage)](./03-helper-scripts-advanced.md):** In-depth guide for `init-package.mjs` and `create-package.mjs`.
  * **[Project Scripts Reference](./04-project-scripts-reference.md):** Comprehensive list of all `package.json` scripts.
* **Development Practices:**
  * **[Branching Strategy](./05-branching-strategy.md):** Recommended Git branching model.
  * **[Commit Conventions](./06-commit-conventions.md):** Guidelines for writing commit messages.
  * **[Testing Guide](./07-testing-guide.md):** How to write and run tests.
  * **[Code Review Process](./08-code-review-process.md):** Expectations for pull requests and reviews.
  * **[Dependency Management](./09-dependency-management.md):** Managing dependencies within the monorepo.
* **[Documentation Development](./10-documentation-development.md):** How to write and maintain project documentation using Docusaurus.
* **[Versioning & Publishing (Changesets In-Depth)](./11-changesets-in-depth.md):** A detailed look at the release process.
* **[CI/CD System Explained](./12-ci-cd-explained.md):** Understanding the GitHub Actions workflows.
* **[Advanced Template Customization](./13-advanced-template-customization.md):** Guides for further tailoring the template to your needs.
* **[Security Practices](./14-security-practices.md):** Tips for keeping your project secure.

## Getting Started with the Template

If you haven't already, make sure to read the main [README.md](https://github.com/lkd0x0is/ts-lib-template/README.md) for initial setup instructions and a quick project overview. This Developer Guide builds upon that foundation.

We encourage you to explore these sections as needed. If you have suggestions for improving this guide, please let us know!
